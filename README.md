## 🛡️ What It Does
Detects fraud in **5 seconds** using AI — fully in **Hindi**!

| Feature | Description |
|---|---|
| 📝 Text/SMS | Fake messages, lottery scams, KYC frauds |
| 🔗 URL Scan | Deep link analysis + Punycode attack detection |
| 📷 Image | Fake screenshots, fake bank notifications |
| 📱 QR Code | Fake UPI QR, hidden phishing links |
| 🎥 Video | Frame-by-frame fraud analysis |

## 🧠 Smart Features
- ✅ **8 AI Models** with automatic fallback (Gemini + Gemma)
